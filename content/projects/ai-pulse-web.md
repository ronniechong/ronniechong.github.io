# AI Pulse

A public dashboard tracking which AI models the world actually uses —
daily rankings, provider market share, geographic adoption, and an
AI-generated daily commentary layer, all fetched live from a versioned
data pipeline that runs entirely on free-tier serverless infrastructure.

Website: **https://ronniechong.com/ai-pulse-web/**

Type `open ai-pulse-web` to navigate to the site.
